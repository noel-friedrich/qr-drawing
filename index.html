Step 1: Set Finder Patterns

Place the three finder patterns that scanners use to locate and orient the QR code.

Step 2: Set Timing Patterns

Place the alternating timing modules between the finder patterns so the grid spacing can be read.

Step 3: Set Error Correction

Choose the QR error correction level and write its format-information indicator bits.

Step 4: Set Format Error Correction

Write the format check bits for the selected error correction level and masking pattern. This also includes the masking pattern indicator, because both values are protected together in the 15-bit QR format information.

Step 5: Set Data

Encode the selected input and fill the complete data-codeword area with payload, terminator, and pad bytes.

Step 6: Set Error Correction

Generate and place the Reed-Solomon error correction codewords for the padded data blocks.

Step 7: Apply Chosen Masking Pattern

Invert data and error-correction modules according to the selected masking pattern.
