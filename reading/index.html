Understand the version

A QR symbol is always square. Its side length determines the version: (size - 17) / 4.

Masking-pattern reference

Bits Pattern Invert when
000 0 (row + column) mod 2 = 0
001 1 row mod 2 = 0
010 2 column mod 3 = 0
011 3 (row + column) mod 3 = 0
100 4 (floor(row / 2) + floor(column / 3)) mod 2 = 0
101 5 (row × column mod 2) + (row × column mod 3) = 0
110 6 ((row × column mod 2) + (row × column mod 3)) mod 2 = 0
111 7 ((row + column) mod 2 + (row × column) mod 3) mod 2 = 0
